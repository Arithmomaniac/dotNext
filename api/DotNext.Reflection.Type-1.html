Equals

Provides smart equality check.

public static readonly Operator<T, T, bool> Equals

Field Value

Operator<T, T, bool>

Remarks

If type T has equality operator then use it. Otherwise, for reference types, this delegate always calls Equals(object, object) method. For value type, it calls equality operator or Equals(T) if it is implemented by the value type; else, it calls Equals<TOther>(in T, in TOther).

GetHashCode

Provides smart hash code computation.

public static readonly Operator<T, int> GetHashCode

Field Value

Operator<T, int>

Remarks

For reference types, this delegate always calls GetHashCode() virtual method. For value type, it calls GetHashCode() if it is overridden by the value type; otherwise, it calls GetHashCode(in T, bool).
