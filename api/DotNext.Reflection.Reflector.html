Unreflect(FieldInfo, BindingFlags, bool)

Creates dynamic invoker for the field.

public static DynamicInvoker Unreflect(this FieldInfo field, BindingFlags flags = BindingFlags.GetField | BindingFlags.SetField, bool volatileAccess = false)

Parameters

field FieldInfo

The field to unreflect.

flags BindingFlags

Describes the access to the field using invoker.

volatileAccess bool

true to generate volatile access to the field.

Returns

DynamicInvoker

The delegate that can be used to access field value.

Remarks

This method doesn't cache the result so the caller is responsible for storing delegate to the field or cache. flags supports the following combination of values: GetField, SetField or both.

Exceptions

ArgumentOutOfRangeException

flags is invalid.

NotSupportedException

The type of field is ref-like value type.
