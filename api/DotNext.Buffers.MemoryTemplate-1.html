Remarks

This type is aimed to fast replacement of the sequence of elements called placeholder in the original sequence of elements. In other words, it is an implementation of find-and-replace algorithm. Pre-compiled template allows to reuse it when rendering with different arguments is required. The rendering process is much faster than Format(string, params object[]) especially for large templates. However, the rendering process doesn't offer formatting procedures.
