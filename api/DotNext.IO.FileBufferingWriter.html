Class FileBufferingWriter

Represents builder of contiguous block of memory that may switch to file as its backing store.

Syntax
public sealed class FileBufferingWriter : Stream, IBufferWriter<byte>, IGrowableBuffer<byte>, IReadOnlySpanConsumer<byte>, ISupplier<ReadOnlyMemory<byte>, CancellationToken, ValueTask>, IFunctional<Func<ReadOnlyMemory<byte>, CancellationToken, ValueTask>>, IResettable, IFlushable
Remarks

This class can be used to write buffered content to a target or IBufferWriter<T>. Additionally, it's possible to get buffered content as . If memory threshold was not reached then returned instance references bytes in memory. Otherwise, it references memory-mapped file.
