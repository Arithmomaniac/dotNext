SuppressCancellation(ValueTask<Holder>)

Suspends cancellation of lock acquisition and converts the canceled operation result into unsuccessfully acquired lock.

[Obsolete("Catch exception manually instead.")]
public static ValueTask<AsyncLock.Holder> SuppressCancellation(this ValueTask<AsyncLock.Holder> result)

Parameters

result ValueTask<AsyncLock.Holder>

The result of the lock acquisition.

Returns

ValueTask<AsyncLock.Holder>

The task representing the lock acquisition.

Remarks

This method is usually combined with AcquireAsync(CancellationToken) or TryAcquireAsync(TimeSpan, CancellationToken) calls to avoid OperationCanceledException if the lock acquisition is already canceled at the time of the call. If the acqusition is canceled then this method returns empty AsyncLock.Holder.
